Attention : avec boostrap, width correspond à offsetWidth.
Margin et border seront intégrés dans la largeur width.
Width est à 424 - padding - border = 400px pour les élément P child du div.
Si l'on souhaite 400 popur un child il faudra donc mettre un width à 424 ou plus selon padding et border
Sans bootstrap, le padding et le border sont bien ajoutés à width. 400 + padding + border.
